<h2 id="started">📌 About </h2>

This is a simple Weather Forecast Application built with HTML, CSS, and JavaScript, using the OpenWeatherMap API to fetch real-time weather data.
With this project, I aimed to improve my skills in API consumption and DOM manipulation, while also working on layout and responsiveness.
Users can search for any city and get current weather information such as:
    🌡️ Temperature
    🌧️ Weather description
    🌇 Location details
